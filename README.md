How to use:
Download and Extract: When you download the repository, please make sure to extract all files from the .zip archive to a folder on your computer. Do not run the files directly from inside the archive, as this may prevent the game from loading correctly.

Launch the Game: Run launcher.exe to start the game.

Select speed lvl: When the launcher starts, it will ask you to enter the number of game instances you wish to open (e.g., entering "10" will launch 10 copies of the game) for example 10 copy are X10 speed and 200 copy are 200X speed (highly recommended to run <100)

Closing the Game: If you close any single game window, the launcher will automatically close all other running instances and also with antivirus auto closer of windows wont work
